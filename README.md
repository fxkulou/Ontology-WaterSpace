# Ontology-WaterSpace
# 本仓库文件对应于《实景三维典型水体空间本体建模》（投稿中）所使用数据。

数据使用遵循CC BY 4.0许可，用于非商业研究。

数据文件包括：

1.本体建模文件（NRFFM_20231216.EAP），该文件可以使用Enterprise Architect软件打开。模型内部结构如图Structure所示包括如下几个部分：

     ·ISO 19152:2012 Land Administration Domain Model 土地管理域模型，该模型用于建模参考；
     ·NRFFM建模包，该包用于自然资源全要素统一概念模型建模（对应论文《自然资源全要素概念模型构建》——发表于中国土地科学）。
                  NRFFM内新建有水体空间包，该包内容对应于水体空间建模内容；
                  塔什库尔干河的模型实例位于该模块下。
     ·NRUM建模包，该包为自然资源全要素模型的英文版（对应论文《Object Analysis and 3D Spatial Modelling for Uniform Natural Resources in China》——发表于Land)；
     ·Old_Basic，被淘汰的基础概念；
     ·Old_RRR，被淘汰的RRR概念；
     ·Sample，基于当前概念模型建立的多个实例（当前为7个）。
2.河流绘制示意图.skp。

      该文件为多种内陆淡水水体空间的利用形态立体绘制示意图。

3.新疆塔什库尔干河流域数据包。（该数据包含含原始数据、过程数据和结果数据）

     ·原始数据：文件夹（新疆塔什库尔干河遥感水文站数据、Google地球数据、现场图）。注：地形数据依据商业软件水经注采集，未经授权不支持开放获取；新疆塔什库尔干河遥感水文站数据为北京师范大学水科学研究院开放数据(https://ecohat.bnu.edu.cn/)。
              新疆塔什库尔干河遥感水文站数据：包括数个观测站点的长期观测数据，研究对其进行了汇总分析（数据汇总表.xls；统计分析表.xls）；
              Google地球数据：包括站点位置、google影像、河流中心线。
              现场图：开放获取的现场调研图像，显示周边环境。
     ·过程数据：文件夹（河流面数据、ArcGIS-ArcScene处理数据），包括河流面数据、河流中心线、河流的三维边界值，以及过程模型（Dae格式）
     ·结果数据：文件夹（制图、Sketchup建模数据），包括过程数据导出至Sketchup建模软件中的结果模型，以及精修制图文件。
